== Description ==


accessi.do is the most enhanced and customized widget that enables accessibility options for any website.


with accessi.do you only need to register your website once and you have it running on your WP Blog.


Enable millions of audiences with disabilities to easily navigate your site, change text sizes, use the keyboard to jump between text blocks and use contrast settings to better reading experience.


As a manager of your widget you have a dashboard that allows you to control  ALT attribute for all images from one place, see the images that need to fix and get amazing report regarding your site images ALT Settings.


accessi.do are WCAG 2.0 authorized and can help you understande the needs in order to have your site also accessible.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
